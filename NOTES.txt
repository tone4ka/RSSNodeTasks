Сценарии ошибок

Вход: пользователь дважды передает один и тот же аргумент cli; 
    Результат: отображается сообщение об ошибке; например ввод: 
    node my_caesar_cli -c C1-C1-A-R0 -c C0
    результат Error: You provided -c argument more than once:;

Ввод: пользователь не передает аргумент -c или --config; 
    Результат: отображается сообщение об ошибке;

Ввод: пользователь передает аргумент -i с несуществующим путем или без доступа для 
чтения; 
    Результат: отображается сообщение об ошибке;

Ввод: пользователь передает аргумент -o с путем к каталогу, который не 
существует или не имеет доступа для чтения; 
    Результат: отображается сообщение об ошибке;

Ввод: Пользователь передает неверные символы в аргумент для --config; 
Результат: отображается сообщение об ошибке;

Сценарии успеха
Вход: пользователь передает правильную последовательность символов в 
качестве аргумента для --config, который соответствует регулярному выражению; 
    Результат: тест пройден
